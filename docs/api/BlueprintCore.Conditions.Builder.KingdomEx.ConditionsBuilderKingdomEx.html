Class ConditionsBuilderKingdomEx

Extension to ConditionsBuilder for conditions involving the Kingdom and Crusade system.

Inheritance
System.Object
ConditionsBuilderKingdomEx
Namespace: BlueprintCore.Conditions.Builder.KingdomEx
Assembly: BlueprintCore.dll
Syntax
public static class ConditionsBuilderKingdomEx
Remarks

Conditions are supported using extension methods. Include the extension namespaces as needed.

When Build(Validator) is called the Kingmaker.ElementsSystem.ConditionsChecker is constructed, validated, and returned. If any errors are detected by Validator they will be logged as a warning.

Do not call Build(Validator) twice on the same builder.

If a method calls for a string to represent any type of blueprint, you can pass the blueprint's Kingmaker.Blueprints.SimpleBlueprint.AssetGuid as a string or as a name you already provided using AddGuidsByName(Dictionary<String, String>).

ConditionsBuilderAreaEx Conditions involving the game map, dungeons, or locations. See also ConditionsBuilderKingdomEx for location related conditions specifically tied to the Kingdom and Crusade system.
ConditionsBuilderBasicEx Most game mechanics related conditions not included in ConditionsBuilderContextEx.
ConditionsBuilderContextEx Most Kingmaker.UnitLogic.Mechanics.Conditions.ContextCondition types. Some Kingmaker.UnitLogic.Mechanics.Conditions.ContextCondition types are in more specific extensions such as ConditionsBuilderKingdomEx.
ConditionsBuilderKingdomEx Conditions involving the Kingdom and Crusade system.
ConditionsBuilderMiscEx Conditions without a better extension container such as game difficulty.
ConditionsBuilderNewEx Conditions defined in BlueprintCore and not available in the base game.
ConditionsBuilderStoryEx Conditions related to the story such as companion stories, quests, name changes, and etudes.
Make a melee attack if the target is in range: 
// Provides the MeleeAttack extension for ActionsBuilder
using BlueprintCore.Actions.Builder.ContextEx; 
// Provides the TargetInMeleeRange extension for ConditionsBuilder
using BlueprintCore.Conditions.Builder.NewEx;

var actionList =
ActionsBuilder.New()
.Conditional(
ConditionsBuilder.New().TargetInMeleeRange(),
ifTrue: ActionsBuilder.New().MeleeAttack())
.build();
